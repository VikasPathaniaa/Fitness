## 🏃‍♂️ Run Locally

1. Clone the project

```bash
  git clone <This project\'s URL>
```

2. Go to the project directory

```bash
  cd Fitness
```

3. Install dependencies

```bash
  npm install
```

4. Start the development server

```bash
  npm run dev
```
